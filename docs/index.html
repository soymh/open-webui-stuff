Memory Enhancement Tool for LLM - Documentation

Version 1.0.0 | Author: mhioi

Functionality Overview

Enable the LLM to utilize advanced memory management functionalities. Below are directives for using the available operations in the Tools class:

Usage Instructions for Tools Class

1. Adding Memories

Command the LLM to store a piece of information in the current memory file:

"Add a memory with tag 'personal' and memo 'Learned a new programming concept', added by LLM"

2. Recalling Memories

Request the LLM to list all the stored memories:

"Recall all memories stored in the current file"

3. Clearing Memories

To erase all memories, confirm twice before instructing the LLM to proceed:

"Clear all memories after confirming"

4. Refreshing Memory

Ask the LLM to optimize and refresh memory entries, including reindexing:

"Refresh and optimize the memory data"

5. Updating Memory Entries

Instruct the LLM to update a specific memory based on its index:

"Update memory at index 3 with tag 'work' and memo 'Finish report by Monday' added by user"

6. Adding Multiple Memories

Guide the LLM to add several memory entries at once:

"Add multiple memories [{'tag': 'personal', 'memo': 'Morning walk', 'by': 'LLM'}, {'tag': 'work', 'memo': 'Team meeting at 10 AM', 'by': 'LLM'}]"

7. Deleting Memory Entries

Command the LLM to delete a specific memory by its index:

"Delete memory at index 5, confirmed by LLM"

8. Deleting Multiple Memories

Request the LLM to remove several memories based on a list of indices:

"Delete memories at indices [2, 4, 6], confirmed by LLM"

9. Switching or Creating Memory Files

Instruct the LLM to switch or create a new memory file to start storing entries:

"Switch to or create a memory file named 'project-notes'"

10. Listing Memory Files

Instruct the LLM to display available memory files:

"List all available memory files"

11. Current Memory File

Ask the LLM which memory file is currently active:

"Which memory file is currently in use?"

12. Deleting a Memory File

Guide the LLM to delete a specified memory file with necessary confirmation:

"Delete the file 'past-notes.json' after confirming twice"

Valves

You need to set Valves in the Tool's Settings inorder to use them

USE MEMORY

This is the most imprtant valve; It allows to use the tool in the chat if set by true.
