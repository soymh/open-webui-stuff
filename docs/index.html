Memory Enhancement Tool for LLM

Version 1.0.0

Author: mhioi

Overview

The Memory Enhancement Tool augments the functionality of a local large language model (LLM) web interface by integrating persistent memory operations. This tool allows the LLM to manage data dynamically through operations such as adding, deleting, updating, and switching among different memory files. Its capabilities enable the LLM to maintain context across interactions, providing a more human-like, memory-enhanced interaction approach.

Commands for LLM

Add Memory

Command the LLM to add a new memory entry to the current memory file. Use the syntax:

"Add a memory tagged as 'T' with the content: 'X' by 'Y'"

Here, 'T' is the tag (e.g., personal, work, etc.), 'X' is the content to store, and 'Y' indicates the source (typically 'user' or 'LLM').

Recall Memory

To retrieve and list all stored memories in the current file, instruct the LLM:

"List all memories from the current file"

The LLM will enumerate each memory entry, providing structured database content for review.

Clear All Memories

Reset all memory entries in the current file. Execute this command:

"Clear all memories in the current file"

Note: This action requires confirmation twice to prevent accidental data loss.

Update Memory

You can refresh an existing memory by index, modifying its tag or content. Command:

"Update memory at index n with tag 'T', content 'M' by 'Y'"

Parameters: n for the index, 'T' for the tag, 'M' for new data, 'Y' for originator.

Delete Memory

To remove a specific memory entry using its index, use:

"Delete memory at index n"

For deleting multiple memories:

"Delete memories at indices n1, n2, ..."

Manage Memory Files

Switch or Create Memory File

Direct the LLM to switch to or create a new memory storage file:

"Switch to or create memory file named 'file_name'"

The LLM will carry out operations in this newly set file going forward.

Delete Memory File

Command to delete a memory file after confirming twice:

"Delete memory file 'file_name'"

The LLM ensures safe file operations, confirming availability and switching files if necessary.

List Memory Files

To inventory all memory files in the operational directory:

"Inventory all available memory files"

This provides a list of recognizable files with a .json extension.

Identify Current Memory File

To discern which memory file is currently in operation:

"Identify the current memory file"

The LLM will notify you of the active memory file name.
