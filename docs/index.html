Memory Enhancement Tool for LLM

Version 1.0.0

Overview

The Memory Enhancement Tool allows the integration of memory functionalities within a large language model (LLM) web UI. It provides seamless memory operations including addition, deletion, update, and switching among multiple memory files. This tool enhances the LLM's capacity to interact with historical data and manage sessions dynamically.

Commands for LLM

Add Memory

To add a new memory entry in the current file, use the command:

"Add a memory of the text: 'X' to current file"

Where 'X' is the text you wish to store. The LLM will categorize and store this under an appropriate tag as per predefined categories.

Recall Memory

To retrieve and view all memories from the current memory file, instruct:

"Show all memories from the current file"

Clear All Memories

To clear all memory entries in the current file, command:

"Clear all memories in the current file"

Note: This operation requires two confirmations to execute.

Update Memory

Updating an existing memory based on its index can be done via:

"Update memory index n with tag 'T' and memo 'M'"

Where n is the index, 'T' the tag, and 'M' the new information to store.

Delete Memory

To delete a specific memory entry by index:

"Delete memory at index n"

Where n signifies the position of the entry in the file. For deleting multiple:

"Delete memories at indices n1, n2, ..."

Memory File Operations

Switch or Create Memory File

To create a new or switch to an existing memory file:

"Switch or create memory file named 'file_name'"

Delete Memory File

To delete a specified memory file:

"Delete memory file 'file_name'"

This requires confirmation twice. It ensures locks at alternative files before deletion if the target is current.

List Memory Files

To list all available memory files present in the directory:

"List all available memory files"

Current Memory File

To know the current memory file in use:

"Show the current memory file"
